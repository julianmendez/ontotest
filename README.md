# [WikiHyperGraph](http://julianmendez.github.io/wikihypergraph/)

### Source code

To clone and compile the project:

```
$ git clone https://github.com/julianmendez/wikihypergraph.git
$ cd wikihypergraph
$ mvn clean install
```

To run the project (after compiling it):
```
$ mvn exec:java
```
